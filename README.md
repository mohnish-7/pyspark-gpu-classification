# pyspark-gpu-classification
## Key Features

* **Distributed ML Pipeline:** Engineered a scalable end-to-end multi-class classification pipeline using **PySpark MLlib** and **Databricks**, optimized for processing large-scale hardware specification datasets.
* **Imbalance Mitigation:** Implemented a robust feature engineering workflow featuring **Pearson correlation analysis** for dimensionality reduction and **custom class weighting** strategies to handle significant label imbalance.
* **High-Performance Benchmarking:** Achieved a **0.92 F1-score** using a Tuned Random Forest Classifier, representing a **20% performance lift** over standard Logistic Regression benchmarks.
* **Experiment Lifecycle Management:** Integrated **MLflow** to track hyperparameters, model versions, and evaluation metrics, ensuring full reproducibility and streamlined model deployment.

## Tech Stack
* **Engine:** PySpark (MLlib)
* **Orchestration:** Databricks, MLflow
* **Languages & Tools:** Python, SQL, Pandas, Matplotlib
